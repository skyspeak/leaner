## Consonant Blends Covered

- **ST, PL, DR, TR, CL, BR** - Phase 1 (Easiest)
- **FL, GL, SL, SP, CR, FR, GR** - Phase 2 (Common)
- **BL, PR, SN, SW, TW** - Phase 3 (Expanding)

## How to Use

### View Online
1. Open `learning_stories.html` in any web browser
2. Click on flashcards to see images
3. Navigate through all 58 pages

### Print to PDF
1. Open `learning_stories.html` in your browser
2. Go to File ‚Üí Print (or Cmd/Ctrl + P)
3. Select "Save as PDF"
4. Print all pages or select specific pages

### Local Server
```bash
python3 -m http.server 8000
```
Then open `http://localhost:8000/learning_stories.html` in your browser.

## Interactive Features

- **Clickable Flashcards**: Click any flashcard to reveal an emoji/image
- **Smooth Animations**: Images fade in with scale animations
- **Hover Effects**: Cards respond to mouse hover

## File Structure

- `learning_stories.html` - Main learning resource (all stories and flashcards)
- `blends_list.md` - Complete list of consonant blends with teaching notes
- `README.md` - This file

## Educational Approach

- Starts with 4-letter words (easier to decode)
- One blend at a time (not overwhelming)
- Visual flashcards for recognition
- Stories provide context and make learning fun
- Common words that children encounter daily
